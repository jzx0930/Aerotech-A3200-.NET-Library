firmwareFilePath (String)
The path to the firmware file to load and commit on the HyperWire card.
Remarks
Loading and committing HyperWire card firmware should be done after loading and committing drive firmware. It might be necessary to restart the PC after loading HyperWire card firmware, call CheckIfPCRestartRequired(String) to see if a PC restart is necessary.
