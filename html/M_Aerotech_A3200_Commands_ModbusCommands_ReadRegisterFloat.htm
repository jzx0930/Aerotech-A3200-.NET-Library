Parameters
RegName (RegisterType)
The register set to access. Either MasterInputWords, SlaveInputWords, MasterOutputWords, or SlaveOutputWords. The input words have read-only access, and the output words have read and write access.
RegNumber (Int32)
The index number of the register to access. This argument can be an expression and must be a positive integer.
Return Value
Single
The stored value.
Remarks

This command is the same as MODBUSREGSINGLE in AeroBasic.
