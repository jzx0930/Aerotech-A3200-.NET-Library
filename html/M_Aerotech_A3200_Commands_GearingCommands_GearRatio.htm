GearRatio Method (SlaveAxis, SlaveValue, MasterValue)
Namespaces ► Aerotech.A3200.Commands ► GearingCommands ► GearRatio(Int32, Int32, Int32)
Defines the ratio of movement between the master axis and slave axis. The slave axis moves SlaveValue counts for every MasterValue counts.

Parameters
SlaveAxis (Int32)
The axis that is set as the slave axis.
SlaveValue (Int32)
The number of counts the slave axis moves for every MasterValue counts of master axis motion.
MasterValue (Int32)
The number of counts of master axis motion for which the slave axis moves SlaveValue counts.
Remarks

This command is the same as GEAR RATIO in AeroBasic.
