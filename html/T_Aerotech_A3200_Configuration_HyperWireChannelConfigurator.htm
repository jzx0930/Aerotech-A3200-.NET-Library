Remarks

To configure HyperWire communication channels, do the following:

  1. Create a new HyperWireChannelConfigurator object to begin the process of assigning HyperWire communication channels. The A3200 controller will stop after you construct this object.
  2. Use the Drives property to access communication information about each connected HyperWire drive.
  3. Set the NewCommunicationChannel property for each connected HyperWire drive. Channels must be unique and must not overlap.
  4. Call the Commit()()()() method to save the communication configuration for all connected HyperWire drives.
  5. Call the Dispose()()()() method to end configuration and clean up any resources.
