Parameters
Task (TaskId)
The task from which to retrieve the status items. To retrieve the status items from the current task, use $taskindex as the value.
DataItem (TaskStatusSignal)
Specifies the task status to retrieve. Table: TASK STATUS Data Items shows the values that you can use.
AdditionalData (Int32)
Additional data for the specified data item. Some data items require additional data. If additional data is not required, specify a value of 0 for this argument.
ControllerVariable (String)
The controller variable on which to retrieve the data. The value of the controller variable must be sufficient to store the number of samples specified by the <SampleRate> argument.
SampleRate (Int32)
The number of samples to collect. This must be 1, 2, 4, 8, 16, 24, or 48.
Remarks

This command is the same as TASKSTATUSFAST in AeroBasic.
