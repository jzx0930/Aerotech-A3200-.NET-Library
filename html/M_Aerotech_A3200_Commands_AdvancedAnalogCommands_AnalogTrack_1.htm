Parameters
Axis (Int32)
The axis from which the controller tracks the servo loop value.
AnalogOutput (Int32)
The analog output to which the servo loop value is sent.
ServoValue (Int32)
Specifies the servo loop value that the controller tracks.
ScaleFactor (Double)
The value that you use to change the servo loop units to the analog output voltage.
OffsetValue (Double)
This value is applied with the tracking value to the analog output. Use this argument if you want to track position on a stage where the position can never be negative. The units are volts.
Remarks

This command is the same as ANALOG TRACK in AeroBasic.
