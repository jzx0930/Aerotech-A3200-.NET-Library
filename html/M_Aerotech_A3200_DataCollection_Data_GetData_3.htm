Remarks

The progress provided by the progressChangedEventHandler, the progress is provided for retrieval of the data, not for collection of data.

This call blocks until the number of specified points have been collected, then retrieves all the points.

The data can be converted to a different set of units, see ConvertUnits(UnitInformation) for more details.
