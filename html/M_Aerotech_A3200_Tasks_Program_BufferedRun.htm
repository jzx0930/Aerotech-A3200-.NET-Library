Remarks

This mode can run programs of arbitrary size and should be used only if the Run(String) cannot run a program of given size. This mode has major limitations in terms of programming language features. If this is used the task cannot be used for anything else until the program execution is done or Stop()()()() is called.

See the main help file for additional information.
