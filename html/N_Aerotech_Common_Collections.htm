Remarks

The collections are organized by whether their contents can change or not, and whether the currently selected objects can change or not.

The NamedCollection<(Of <(<'TObject, TName>)>)> allows modifying its contents, while NamedConstantCollection<(Of <(<'TObject, TName>)>)> does not allow to change the elements it contains.

The NamedConstantCollectoin does not allow to hide elements it contains, while the NamedMaskedConstantCollection<(Of <(<'TObject, TName, TMask>)>)> has some elements masked out, and NamedMaskableCollection allows to sepecify which elements to hide and which ones not to hide.
