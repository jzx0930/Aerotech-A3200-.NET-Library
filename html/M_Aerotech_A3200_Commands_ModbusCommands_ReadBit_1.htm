Parameters
ConnectionNumber (Int32)
Identifies the Modbus connection to use. Specify the axis name on which to use the Modbus register set. Specify a number from 0 - 15 to use the corresponding PC Modbus register set (1 - 15 only available with the Professional option).
BitName (RegisterType)
The bit set to access. Either MasterInputBits, SlaveInputBits, MasterOutputBits, or SlaveOutputBits. The input bits have read-only access, and the output bits have read and write access.
BitNumber (Int32)
The bit number in the set. This argument can be an expression and must be a positive integer.
Return Value
Int32
The stored value.
Remarks

This command is the same as MODBUSBIT in AeroBasic.
