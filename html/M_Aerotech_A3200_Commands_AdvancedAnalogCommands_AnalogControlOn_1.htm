Parameters
Axis (Int32)
The axis on which to execute the command
AnalogInput (Int32)
The analog input to use as the input to this command.
ScaleFactor (Double)
The value that you use to scale the analog input voltage. The units are user units per volt.
OffsetValue (Double)
The value that specifies the voltage offset for the analog input signal. The units are volts.
SpeedValue (Double)
Specifies the maximum speed at which the controller commands the axis to move while the axis is under the control of this command. The units are user units per second. If you do not specify this argument or if you specify a value of 0 for this argument, then the controller does not limit the maximum speed.
Remarks

This command is the same as ANALOG CONTROL ON/OFF in AeroBasic.
